Created widgets for live chat and Knowledge based questions  on Freshdesk and connected it to a website I created using Blogspot.

Steps>
Create a widget on Freshdesk (Admin>Channels>Web chat>Create widget>New widget/New web chat topic)
Create a website on Blogspot
Go back to widget you created on Freshdesk
Go to deploy code
Copy the code
Go to Blogspot (Layout>Add a Gadget>Html/JavaScript>Title>Paste the link on the Content>Save)
